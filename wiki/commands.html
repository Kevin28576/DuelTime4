指令總覽

主指令：/dueltime4（別名：/dueltime、/dt、/dt4）

玩家常用指令

指令 用途
/dt help查看玩家指令說明
/dt start開啟匹配面板
/dt join <arena_id>加入指定競技場
/dt send <player> [arena_id]發送對戰邀請
/dt accept [player]接受邀請（GUI fallback）
/dt decline [player]拒絕邀請（GUI fallback）
/dt queue sound [on|off|toggle|status]控制個人隊列音效提醒
/dt queue cooldown [player]查詢隊列冷卻（查他人需管理權限）
/dt stats [player]查看玩家數據
/dt record開啟戰績 GUI
/dt rank me查看個人排名
/dt level me查看個人等級
/dt point me查看個人點數
/dt shop開啟點數商店
/dt spectate <arena_id>觀戰
/dt quit離開目前對戰
/dt lang [file]切換語言檔

管理員指令（dueltime.admin）

指令 用途
/dt adminhelp查看管理指令說明
/dt reload重載設定與語言
/dt stop [reason]中止進行中的對戰
/dt doctor [all|services|database|queue]健康檢查
/dt queue debug隊列診斷
/dt update check|download|status更新器操作
/dt balance view <player>查看玩家點數
/dt balance set <player> <value>設定玩家點數
/dt balance config view查看可線上調整參數
/dt balance config set <key> <value>線上調整平衡設定
/dt record export [player] [json|csv] [limit]匯出戰報
