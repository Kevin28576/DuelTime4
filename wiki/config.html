設定檔重點

主檔案：plugins/DuelTime4/config.yml

主要區塊

區塊 用途
Message前綴與預設語言
System.restart-protection重啟保護機制
DatabaseMySQL / SQLite 模組儲存配置
Arena.classic經典對戰獎勵、回傳、連勝連敗
Arena.classic.matchmaking.queue-sound隊列音效提醒與玩家自訂
Arena.classic.matchmaking.watchdog隊列看門狗
Arena.classic.matchmaking.leave-penalty離場懲罰
Updater版本檢查、自動下載、SHA-256 驗證
Network.discord-webhookWebhook 通知（Embed）
Ranking排行榜刷新與全息
Level等級與階級曲線

匹配關鍵設定

Arena:
  classic:
    matchmaking:
      confirm-timeout: 15
      queue-sound:
        enabled: true
        allow-player-toggle: true
        interval-seconds: 5
      watchdog:
        enabled: true
        interval-seconds: 3
      leave-penalty:
        enabled: true
        apply-on-quit-command: true
        apply-on-disconnect: true
        apply-point-deduction: true
        point: 2
        apply-queue-cooldown: true
        cooldown: 20

建議watchdog.interval-seconds 設在 2~5 秒。

更新器設定

Updater:
  enabled: true
  check-on-startup: true
  auto-download: false
  verify-sha256: true
  manifest-url: "https://.../network/update.yml"
  fallback-version-url: "https://.../network/version"
  download-url-template: "https://github.com/Kevin28576/DuelTime4/releases/download/v{version}/Dueltime4-Bukkit-{version}.jar"

下載路徑要使用 v{version} 格式，例如 v4.1.3。

語言檔建議

  • 主語言集中於 zh_tw.yml。
  • 可調訊息優先放在 Dynamic.*。
  • Webhook 的 content 可留空，也可自訂 TAG/附加文字。

線上調整（免重啟）

可透過 /dt balance config set 微調參數。

/dt balance config set win-point 2
/dt balance config set leave-penalty-point 3
/dt balance config set queue-sound-enabled true
/dt balance config set watchdog-interval-seconds 3
