更新重點

近期 4.1.x 版本摘要。

4.1.3

新增
  • /dt doctor 健康檢查（services / database / queue）。
  • /dt queue debug 隊列診斷面板。
  • /dt record export [player] [json|csv] [limit] 戰報匯出。
改善
  • ActionBar 隊列資訊更完整（秒數、人數、預估時間）。
  • 匹配確認 GUI 與 fallback 指令流程穩定化。
  • 排隊音效提醒可由玩家自行開關。

4.1.2

新增
  • 更新器支援 manifest、下載連結、SHA-256 驗證。
  • /dt update check|download|status 管理指令。
  • Webhook Embed 支援語言檔配置，並保留可自訂 content。
調整
  • 打包檔名固定：Dueltime4-Bukkit-{version}.jar。
  • 下載路徑採用 v{version} tag 格式。
